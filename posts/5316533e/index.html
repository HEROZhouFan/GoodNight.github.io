生成密钥 SSH key

按照提示完成三次回车，即可生成 ssh key

找到生成的 .pub 文件

/Users/zhoufan/.ssh/id_rsa.pub（隐藏文件 shift+command+. 显示）

登录github账号，进入到Settings

New SSH key

取一个名字，将.pub 的密钥粘贴进来

这样就配置好了 SSH 访问 Github 的权限
